# Evidence Context Effects

## Overview

Audit context/performance claims with primary studies, numbers, effect sizes or ranges, and a broader causal model. Do not only debunk anecdotes; answer the user's real conceptual question.

## Required Output

1. Reconstructed question.
2. Numbers/ranges from the primary source when available.
3. What the anecdote/study supports.
4. What is overclaimed or confounded.
5. Human evidence beyond the anecdote.
6. Causal model for environment/history effects.
7. Practical implication and uncertainty.
8. Saved appendix if complex.

## Workflow

1. Find the original study or closest primary source.
2. Extract the actual setup, numbers, range, sample, and conditions.
3. Compare those numbers to the viral/user version.
4. Identify confounds and whether the named mechanism was measured.
5. Search for human evidence: self-efficacy, controllability, role models, social norms, deliberate practice, stereotype threat, feedback loops, stress, flow.
6. Estimate magnitude: short-term effects, extreme stress/endurance effects, and long-term compounding effects.
7. State the strongest defensible claim.

## Quality Gates

- Did we give the requested numbers/range?
- Did we answer the broader environment/history question?
- Did we distinguish animal/anecdote evidence from human evidence?
- Did we avoid motivational overclaim?
- Did we include practical implications?
