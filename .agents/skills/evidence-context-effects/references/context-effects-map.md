# Context Effects Map

Use as a source checklist, not as a claim of settled truth.

## Source Families

- Primary study behind the anecdote.
- Learned helplessness / controllability.
- Self-efficacy and performance.
- Role-model attainability.
- Stereotype threat and identity/context cues.
- Deliberate practice and feedback.
- Flow / challenge-skill balance.
- Stress, sleep, threat, scarcity, and cognitive bandwidth.
- Social network and local norm effects.

## Magnitude Framing

- Short-term cognitive effects: often small-to-medium; task dependent.
- Stress/endurance/persistence: can be large under extreme conditions.
- Long-term environment effects: can be very large through selection, repeated practice, mentors, norms, and opportunity access.

Always say which magnitude claim is supported by which evidence family.
