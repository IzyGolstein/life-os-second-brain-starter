# Career Source Mix

Use this to avoid one-source answers.

## Tool Access

- Official supported-country docs.
- Provider terms/status pages.
- Habr/vc.ru/news/community reports.
- GitHub issues or developer forums when tooling behavior is reported by users.
- Account-blocking anecdotes must be labeled as anecdotal unless backed by provider policy.

## Career Paths

- Program pages for PhD/master's/fellowships.
- Lab pages and advisor publications.
- Job postings and company career pages.
- Salary sources by country and role.
- Immigration/visa official pages where decisions depend on eligibility.

## Output Lists

Keep main answer short. Save appendix tables with:

- country;
- target;
- role/program;
- link;
- fit;
- blockers;
- deadline or search cadence;
- salary/funding note when relevant.
