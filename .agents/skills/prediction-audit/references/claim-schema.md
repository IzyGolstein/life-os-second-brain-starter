# Claim Schema

Minimum fields:

- `claim_id`
- `source_work`
- `chapter_or_location`
- `short_quote_anchor`
- `paraphrase`
- `claim_type`
- `forecast_target`
- `resolution_date`
- `geography`
- `indicator`
- `baseline`
- `evidence_urls`
- `status`
- `score_0_4`
- `specificity`
- `difficulty`
- `beat_baseline`
- `auditor_note`

Rubric:

- `4`: accurate and specific.
- `3`: directionally right but imprecise.
- `2`: mixed or ambiguous.
- `1`: mostly wrong.
- `0`: clearly wrong.
